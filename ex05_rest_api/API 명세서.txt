API 명세서

회원등록
  요청 : POST /api/members
  요청 본문 : {"email" : "user@example.com" }
  응답 본문 :  등록된 회원 조회(MemberResponse)
  상태 코드 : 201 Created

회원 전체 조회
  요청 : GET / api/memebers
  요청 본분 : x
  응답 본문 : 회원목록 (List<membersResponse>)
  상태 코드 : 200 OK

회원 단건 조회
요청 : GET / api/members/{id}
요청 본분 : x
응답 본문 : 단건 회원 정보(MemberResponse)
상태 코드 : 200 Ok /  404 Not Found

회원 수정
요청 : PUT / api/memebers/{id}
요청 본문 : {"email": "user@example.com"}
응답 본문 : 수정된 회원 정보(MemberResponse)
상태 코드 : 200 OK

회원 삭제
요청 : DELETE / api/memebers/{id}
요청 본문 : x
응답 본문 : x
상태 코드 : 204 No Content
